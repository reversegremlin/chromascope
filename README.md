# Chromascope: Audio-Reactive Particle Universe
## High-Energy Physics & Sacred Geometry Renderers

Chromascope is a project dedicated to translating sound into organic, high-intensity particle simulations. It combines fluid dynamics, fractal geometry, and radioactive decay physics into a single, audio-reactive visual suite.

The **Radioactive Cloud Chamber** renderer is a Wilson-style simulation that captures the beauty of atomic decay.

*   **Dual-Buffer Vapor Engine:** High-intensity Ionization Tracks (Alpha/Beta/Gamma) and persistent, drifting Smoke Buffers.
*   **Dynamic Symmetrical Mirroring:** Two independent radioactive worlds ("Matter" and "Antimatter") slide symmetrically along a locked axis.

*   **Cinematic Slow-Sweep:** Smooth, sub-pixel interpolated motion that follows the musical energy.
